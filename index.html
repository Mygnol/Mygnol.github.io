DeceasedCraft: A Modern Zombie Apocalypse

A Minecraft Modpack
JSON/AppsScript


Perhaps the largest non-professional project I've joined to date, DeceasedCraft is a collection of curated mods, configured and combined together to provide a different and unique spin on traditional Minecraft gameplay. While not the main developer in this team of 3, my contributions are predominantly in ensuring the in-game tutorial is clear and concise for new players, and providing automation tools to aid in converting large spreadsheets to dozens of individual JSON configuration files. On the side, I also manage the Discord community of the modpack, general server layout, as well as handle filtering user suggestions and configuring the server bots.
As of writing, we are preparing to port the project to a newer Minecraft version, which will bring with it another major update, tentatively scheduled for sometime next year.

The Weight Of The Crown

Puzzle Platformer
Unity Engine


The latest SEA Game Jam submission. While unfortunately submitted just barely too late to make the deadline, it remains the most polished of my Unity game jam submissions, and contributed heavily to the current suite of Unity code snippets I share across jams. My contributions to this are split almost straight down the middle; all the art, lighting and UI mechanics were done by me.
We also managed to fit in a trailer, which is rare for a game jam game!
